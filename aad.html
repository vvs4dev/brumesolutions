Azure AD

Azure AD is Microsoft’s multi-tenant, cloud based directory and identity management service that combines core directory services, advance identity governance and application access management.



Benefits

Enhanced AD Security

  1. Cloud App Security

  2. Device writeback

  3. MFA

  4. Conditional Access

Minimize IT burden

  1. Reduce operational cost by abstracting hardware details

  2. Keep track of outages or Issues via Connect health

  3. With MIM(MS Identity Manager), customer manage policies and access within their organization while supporting new platform

  4. MDM Auto Enrollment- With Intune

Why Customer opt for Azure AD

Challenges

Multiple location

  1. Customer had multiple locations hence managing identities was a tedious task

  2. Pushing policies was another overhead

Security

  1. Customers seeking for solution which gives the capability of MFA

  2. Customer needed to give their employee only under certain circumstances to few application

  3. Top management should able to reset password without interference of on premise IT

  4. Intricacies in managing group policies, OUs

Solutions/Approach

  1. Azure AD & Domain Services

  2. Conditional Access
